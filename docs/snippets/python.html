import logging

# create logger
logger = logging.getLogger(__name__)
# set log level for all handlers to debug
logger.setLevel(logging.INFO)

# example usage
logger.debug('debug message')
logger.info('info message')
logger.warning('warn message')
logger.error('error message')
logger.critical('critical message')

The previous code returns this:

WARNING:__main__:warn message
ERROR:__main__:error message
CRITICAL:__main__:critical message

Random 16-character password‚Äã

import os
import random
import tempfile

# Create a temporary file
with tempfile.NamedTemporaryFile(delete=False) as f:
    # Write a random 16-character password to the file.
    password = ''.join(random.choice('123456789abcdefghijklmnopqrstuvwxyz') for _ in range(16))
    f.write(password.encode())

    # Flush the file to make sure the data is written immediately.
    f.flush()

    # Get the path of the temporary file.
    temp_file_path = f.name

# Print the path of the temporary file.
print(temp_file_path)

Virtual environment‚Äã

Create a new Python virtual environment. On Linux/macOS, the command below creates an environment in .venv, activates it, and install requirements.

python -m venv .venv
source .venv/bin/activate
pip3 install -r test_requirements.txt

After you finish working deactivate your virtual environment by simply executing the below command.

deactivate

Setting up jupyter environment‚Äã

Run Jupyter server and kernel inside the conda environment‚Äã

Create and start the environment using the following commands.

conda env create -f /Users/boisalai/Documents/Learning/books/AAAMLP/environment_osx.yml
conda activate ml
conda install jupyter
jupyter notebook

To deactivate an active environment, use

conda deactivate

To delete an environment and everything in it, use

conda env remove --name ml

Create ipython kernel for the conda environment‚Äã

conda env create -f /Users/boisalai/Documents/Learning/books/AAAMLP/environment_osx.yml   # creates new virtual env
conda activate ml   # activate environment in terminal
conda install ipykernel   # install Python kernel in new conda env
ipython kernel install --user --name=ml-kernel   # configure Jupyter to use Python kernel

conda deactivate          # this step can be omitted by using a different terminal window than before
conda install jupyter     # optional, might be installed already in system e.g. by 'apt install jupyter' on debian-based systems
jupyter notebook          # run jupyter from system

from pytube import YouTube

def Download(link):
    youtubeObject = YouTube(link)
    youtubeObject = youtubeObject.streams.get_highest_resolution()
    try:
        youtubeObject.download()
    except:
        print("An error has occurred")
    print("Download is completed successfully")
