Phi-2

Phi-2 is a 2.7B parameter language model released by Microsoft with performance that rivals much larger models. It was trained on a mixture of GPT-4 outputs and clean web text.

Phi-2 efficiently runs on Apple silicon devices with 8GB of memory in 16-bit precision.

mkdir demo
cd demo

# Create a virtual environment
python -m venv venv
source venv/bin/activate
pip install --upgrade pip

# Clone the repository
git clone https://github.com/ml-explore/mlx-examples.git
cd mlx-examples/llms/phi2
pip install -r requirements.txt

# Install required packages for faster downloads
pip install huggingface_hub hf_transfer

# Download model
export HF_HUB_ENABLE_HF_TRANSFER=1
huggingface-cli download --local-dir-use-symlinks False --local-dir phi-2 microsoft/phi-2

# Run example
python generate.py --model phi-2 --prompt "Qui est Albert Einstein?" --max-tokens 1000

For more information, see:
