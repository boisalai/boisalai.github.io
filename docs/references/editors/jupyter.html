Create a kernel with a virtual environment​

Setting up a virtual environment for Python and configuring it as a Jupyter kernel.

python -m venv .venv
source .venv/bin/activate   # Activate the virtual environment on MacOS and Linux
pip install --upgrade pip
pip install -r requirements.txt
python3 -m pip install "notebook<7" ipykernel
python -m ipykernel install --user --name=GLO-7030
jupyter notebook
Error when launching jupyter

When launching Jupyter, you may get a blank page with only the jupyter logo, and this terminal error message HTTP 404: Not Found (Schema not found: /opt/homebrew.... See Missing or misshapen translation settings schema for a solution.

After you finish working, deactivate the virtual environment by executing the following command:

deactivate

To remove the kernel from Jupyter, execute the following command:

jupyter kernelspec uninstall GLO-7030

To delete the virtual environment, execute the following command:

rm -rf .venv

Using Pipenv for Environment Management and Dependency Handling​

Use pipenv to ensure more integrated and simpler management of your project's virtual environment and dependencies.

pip install --user pipenv      # Install pipenv if not already installed
pipenv install                 # Create the virtual environment and install dependencies
pipenv install -dev jupyter ipywidgets   # Install or update jupyter and ipywidgets
pipenv run python -m ipykernel install --user --name=GLO-7030   # Install the Jupyter kernel for this environment
pipenv run jupyter notebook    # Launch Jupyter Notebook

When you are done working in your virtual environment, you can exit it easily with pipenv. Unlike the traditional venv method, which uses the deactivate command, pipenv environments are activated and deactivated automatically when running commands with pipenv run. However, if you have spawned a sub-shell in your environment using pipenv shell, you can exit it by typing exit.

Remove the current virtual environment with this command.

pipenv --rm

Adding and removing virtual environments to Jupyter notebook​

## Create the virtual environment
conda create -n my-project-env python=3.9.12

## Activate the virtual environment
conda activate my-project-env

## Make sure that ipykernel is installed
pip install --user ipykernel

## Add the new virtual environment to Jupyter
python -m ipykernel install --user --name=my-project-env

## To list existing Jupyter virtual environments
jupyter kernelspec list

## To list existing conda environments
conda env list

## To remove conda environment
conda env remove -n my-project-env

## To remove the environment from Jupyter
jupyter kernelspec uninstall my-project-env
