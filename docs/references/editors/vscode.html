Visual Studio Code

Starting a new project​

Begin by creating a new project folder using the terminal. Once created, navigate into your new project folder and open it in Visual Studio Code (VS Code).

In the terminal, execute the following commands in the terminal:

mkdir my-project           # Creates a new directory named 'my-project'
cd my-project              # Changes the current working directory to 'my-project'
python3 -m venv .venv      # Create a virtual environment on MacOS and Linux.
source .venv/bin/activate  # Activate the virtual environment.
.venv/bin/python3 -m pip install --upgrade pip  # Upgrade pip.

Create a new Python script file. Name it hello.py and open it with VS Code

touch hello.py
code .

Insert the following script into `hello.py:

import sys
from prefect import flow, task


@task
def display_info():
    print("Current Python virtual environment:", sys.prefix)
    print("Current Python interpreter:", sys.executable)
    print("Python version:", sys.version)
    print("Python search path:", sys.path)


@flow
def main():
    display_info()


if __name__ == "__main__":
    main()

In the VS Code terminal, execute the following command to run the script:

python3 -m pip install prefect

Now run the script. This script should return something like this:

Starting a new project with Pipenv​

If you haven't already installed Pipenv, you can do so by running:

pip3 install pipenv

If you don't already have a Pipfile for your project, initialize one with:

pipenv --python 3.8

In the terminal, execute the following commands in the terminal:

mkdir my-project           # Creates a new directory named 'my-project'
cd my-project              # Changes the current working directory to 'my-project'
pipenv install -d black    # Create a virtual environment and install dependencies. 
pipenv run pip install 'deepspeed<=0.9.3'

After installation, you can check that the package has been installed correctly by using:

pipenv run pip freeze

Create a new Python script file. Name it hello.py and open it with VS Code

touch hello.py  # Create a new Python script file.
code .          # Open the project in VS Code.

Insert the following script into `hello.py:

import sys
from prefect import flow, task


@task
def display_info():
    print("Current Python virtual environment:", sys.prefix)
    print("Current Python interpreter:", sys.executable)
    print("Python version:", sys.version)
    print("Python search path:", sys.path)


@flow
def main():
    display_info()


if __name__ == "__main__":
    main()

In the VS Code Command Palette, select the Python: Select Interpreter command and select the Python PipEnv interpreter.

In the VS Code terminal, execute the following command to run the script:

pipenv shell           # Activates the virtual environment created by pipenv
pipenv install prefect # Installs the 'prefect' package within the virtual environment
black hello.py         # Formats the 'hello.py' script using the Black code formatter
python3 hello.py       # Runs the 'hello.py' script using Python 3

This sequence of commands sets up a Python virtual environment, installs necessary packages, formats your script for readability, and then executes it.

cat Pipfile
[[source]]
url = "https://pypi.org/simple"
verify_ssl = true
name = "pypi"

[packages]
prefect = "*"

[dev-packages]
black = "*"

[requires]
python_version = "3.9"

The Pipfile displayed here is part of the pipenv environment. It specifies the source for package installations, the main and development packages required, and the Python version that this environment is compatible with. Essentially, it's a more structured and comprehensive version of the traditional `requirements.txt`` used in Python projects.

Open VS Code from Chrome​

Just type vscode.dev to start VS Code from Chrome.

Open VS Code from terminal​

  • Open the command palette.
  • Next, into the command palette type shell and hit Enter.
  • Select Shell Command: Install 'code' command in PATH and press Enter. Once you hit enter, voilà! You're all done.
  • Once your terminal is open, you can open VS Code from it by typing $ code . then hit enter.
