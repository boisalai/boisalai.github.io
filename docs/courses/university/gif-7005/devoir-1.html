D1Q1 - Estimateur statistique​

Contexte​

Soit la loi exponentielle, pour laquelle la densité de probabilité est définie par l'équation suivante:

p(x)={λexp⁡(−λx) si x≥00 si x<0p(x) = \begin{cases} \lambda \exp(-\lambda x) \text{ si } x \ge 0\\ 0 \text{ si } x < 0 \end{cases}p(x)={λexp(−λx) si x≥00 si x<0​

L'espérance mathématique de cette loi est E[x]=1λ\text{E}[x]=\frac{1}{\lambda}E[x]=λ1​ alors que la variance est Var(x)=1λ2\text{Var}(x)=\frac{1}{\lambda^2}Var(x)=λ21​.

D1Q1A Question​

Trouver l'expression mathématique de l'estimateur de λ\lambdaλ selon un maximum de vraisemblance.

D1Q1A Solution​

Pour trouver l'estimateur de λ\lambdaλ par la méthode du maximum de vraisemblance, procédons par étapes :

Étape 1. Fonction de vraisemblance​

La fonction de vraisemblance représente la probabilité d'observer l'ensemble X\mathcal{X}X sous l'hypothèse d'un certain paramètre λ\lambdaλ. Ici, nous définissons la vraisemblance de NNN échantillons xt\textbf{x}^txt indépendants tirés d'une distribution exponentielle :

l(λ∣X)≡p(X∣λ)=∏t=1Np(xt∣θ)=∏t=1Nλexp⁡(−λxt)l(\lambda | \mathcal{X}) \equiv p(\mathcal{X} | \lambda) = \prod_{t=1}^N p(\textbf{x}^t|\theta) = \prod_{t=1}^{N} \lambda \exp(-\lambda \textbf{x}^t)l(λ∣X)≡p(X∣λ)=t=1∏N​p(xt∣θ)=t=1∏N​λexp(−λxt)

Étape 2: Log-vraisemblance​

En prenant le logarithme naturel des deux côtés, la fonction log-vraisemblance est donc :

L(λ∣X)≡log⁡l(λ∣X)=∑t=1Nlog⁡(λexp⁡(−λxt))=∑t=1N[log⁡(λ)−λxt]=Nlog⁡(λ)−λ∑t=1Nxt\begin{split} L(\lambda | \mathcal{X}) \equiv \log l(\lambda | \mathcal{X}) & = \sum_{t=1}^N \log(\lambda \exp(-\lambda \textbf{x}^t)) \\ & = \sum_{t=1}^{N} [\log(\lambda) - \lambda \textbf{x}^t] \\ & = N \log(\lambda) - \lambda \sum_{t=1}^{N} \textbf{x}^t \end{split}L(λ∣X)≡logl(λ∣X)​=t=1∑N​log(λexp(−λxt))=t=1∑N​[log(λ)−λxt]=Nlog(λ)−λt=1∑N​xt​

Étape 3: Maximiser la log-vraisemblance​

Pour trouver la valeur de λ\lambdaλ qui maximise la log-vraisemblance, dérivons par rapport à λ\lambdaλ et égalons à zéro :

dl(λ∣X)dλ=0⇒ddλ(Nlog⁡(λ)−λ∑t=1Nxt)=0⇒Nλ−∑t=1Nxt=0⇒Nλ=∑t=1Nxt⇒λ=N∑t=1Nxt\begin{split} \frac{dl(\lambda | \mathcal{X})}{d\lambda} = 0 & \Rightarrow \frac{d}{d\lambda} \left( N \log(\lambda) - \lambda \sum_{t=1}^{N} \textbf{x}^t \right) = 0 \\ & \Rightarrow \frac{N}{\lambda} - \sum_{t=1}^N \textbf{x}^t = 0 \\ & \Rightarrow \frac{N}{\lambda} = \sum_{t=1}^N \textbf{x}^t \\ & \Rightarrow \lambda = \frac{N}{\sum_{t=1}^N \textbf{x}^t} \end{split}dλdl(λ∣X)​=0​⇒dλd​(Nlog(λ)−λt=1∑N​xt)=0⇒λN​−t=1∑N​xt=0⇒λN​=t=1∑N​xt⇒λ=∑t=1N​xtN​​

Ainsi, l'estimateur du maximum de vraisemblance pour λ\lambdaλ est :

λ^=N∑t=1Nxt\hat{\lambda} = \frac{N}{\sum_{t=1}^N \textbf{x}^t}λ^=∑t=1N​xtN​

où λ^\hat{\lambda}λ^ est l'estimateur de maximum de vraisemblance de λ\lambdaλ.

D1Q1B Question​

Déterminez si cet estimateur de λλλ est biaisé dans le cas général?

Indice : Vous pouvez considérer que E[1/x]≠1/E[x]\text{E}[1/x] \ne 1/\text{E}[x]E[1/x]=1/E[x] dans le cas général.

D1Q1B Solution​

Pour déterminer si l'estimateur λ^\hat{\lambda}λ^ est biaisé, nous devons comparer son espérance à la vraie valeur de λ\lambdaλ.

Si l'espérance de l'estimateur est égale à λ\lambdaλ, alors l'estimateur est non biaisé. Sinon, il est biaisé.

L'estimateur λ^\hat{\lambda}λ^ est donné par :

λ^=N∑t=1Nxt\hat{\lambda} = \frac{N}{\sum_{t=1}^N x^t}λ^=∑t=1N​xtN​

Prenons l'espérance des deux côtés :

E[λ^]=E[N∑t=1Nxt]\text{E}[\hat{\lambda}] = \text{E}\left[\frac{N}{\sum_{t=1}^N x^t}\right]E[λ^]=E[∑t=1N​xtN​]

L'espérance de la somme ∑t=1Nxt\sum_{t=1}^N x^t∑t=1N​xt est E[∑t=1Nxt]=NE[x]=Nλ\text{E}[\sum_{t=1}^N x^t]=N\text{E}[x]=\frac{N}{\lambda}E[∑t=1N​xt]=NE[x]=λN​, car E[x]=1λ\text{E}[x]=\frac{1}{\lambda}E[x]=λ1​ pour une distribution exponentielle.

Cependant, nous voulons trouver E[1/∑t=1Nxt]\text{E}\left[1 / \sum_{t=1}^N x^t \right]E[1/∑t=1N​xt] et pas simplement l'espérence de la somme.

Comme mentionné dans l'indice, E[1/x]≠1/E[x]\text{E}[1/x] \ne 1/\text{E}[x]E[1/x]=1/E[x]. En utilisant cet indice, nous pouvons dire que :

E[1/∑t=1Nxt]≠1/E[1∑t=1Nxt]=1NE[x]=1N/λ=λN\text{E}\left[1 / \sum_{t=1}^N x^t \right] \ne 1 / \text{E}\left[\frac{1}{\sum_{t=1}^N x^t}\right] = \frac{1}{N\text{E}[x]} = \frac{1}{N/\lambda} = \frac{\lambda}{N}E[1/t=1∑N​xt]=1/E[∑t=1N​xt1​]=NE[x]1​=N/λ1​=Nλ​

En utilisant cet indice, pour pouvons affirmer que E[λ^]≠λ\text{E}[\hat{\lambda}] \ne \lambdaE[λ^]=λ, indiquant que λ^\hat{\lambda}λ^ est biaisé.

Probabilité a posteriori P(Ci∣x)P(C_i|\mathbf{x})P(Ci​∣x)​

Fonction de décision minimisant le risque avec l'option de rejet​

Pour minimiser le risque en prenant une décision, nous voulons minimiser le coût attendu associé à cette décision. Étant donné les coûts fournis dans l'énoncé :

  • Si nous décidons que x\textbf{x}x appartient à la classe CiC_iCi​ et que c'est correct, le coût est 0.
  • Si nous décidons que x\textbf{x}x appartient à la classe CiC_iCi​ mais que c'est incorrect (i.e., il appartient à une autre classe CjC_jCj​), le coût est 1.
  • Si nous décidons de rejeter x\textbf{x}x, le coût est λ∈[0,1]\lambda \in [0, 1]λ∈[0,1].

Pour chaque observation x\textbf{x}x, la fonction de décision est la suivante :

  1. Calculons les probabilités a posteriori pour toutes les classes, P(Cj∣x)P(C_j|\textbf{x})P(Cj​∣x) pour chaque jjj.
  2. Si la probabilité a posteriori la plus élevée est P(Cj∣x)P(C_j|\textbf{x})P(Cj​∣x) et P(Cj∣x)>1−λP(C_j|\textbf{x}) > 1- \lambdaP(Cj​∣x)>1−λ, alors choisissons la classe CjC_jCj​.
  3. Si aucune classe n'a une probabilité a posteriori supérieure à 1−λ1-\lambda1−λ, rejetons l'observation.
  4. Sinon, choisissez la classe avec la probabilité a posteriori la plus élevée.

La décision pour une observation x\textbf{x}x est :

deˊcision(x)={argmaxjP(Cj∣x) si max⁡jP(Cj∣x)>1−λ,∀j=1,…,KR autrement (rejet de l’observation)\text{décision}(\textbf{x}) = \begin{cases} \text{argmax}_j P(C_j|\textbf{x}) & \text{ si } \max_j P(C_j|\textbf{x}) > 1- \lambda, \forall j=1,\ldots,K \\ R & \text{ autrement (rejet de l'observation)} \end{cases}deˊcision(x)={argmaxj​P(Cj​∣x)R​ si maxj​P(Cj​∣x)>1−λ,∀j=1,…,K autrement (rejet de l’observation)​

La variance pour chaque classe est :

σc2=1Nc∑i=1Nc(xi−μc)2\sigma_c^2 = \frac{1}{N_c}\sum_{i=1}^{N_c} (x_i-\mu_c)^2σc2​=Nc​1​i=1∑Nc​​(xi​−μc​)2

En utilisant NumPy, cela se traduit par :

  • Pour la moyenne : numpy.mean(X_c, axis=0)
  • Pour la variance : numpy.var(X_c, axis=0)

Pour fit(self, X, y)

  • Identifie d'abord les classes uniques dans ton ensemble de données y. Utilise numpy.unique
  • Calcule la probabilité a priori pour chaque classe, soit la proportion de chaque classe dans l'ensemble d'entraînement.
  • Initialise deux matrices pour stocker les moyennes et les variances de chaque classe.
  • Pour chaque classe, filtre les observations appartenant à cette classe, calcule leur moyenne et leur variance, puis stocke ces valeurs.

Pour predict_proba(self, X)

  • Initialise une matrice pour stocker les probabilités a posteriori pour chaque échantillon.
  • Pour chaque classe, utilise la fonction de densité de probabilité d'une distribution normale pour calculer la log-vraisemblance.
  • Convertis ensuite ces valeurs logarithmiques en probabilités réelles.
  • Multiplie par les probabilités a priori, et divise par l'évidence pour obtenir la probabilité a posteriori pour chaque classe.

Pour predict(self, X)

  • Utilise predict_proba pour obtenir les probabilités a posteriori.
  • Pour chaque échantillon, vérifie si la probabilité maximale est inférieure à 1 - _lambda.
  • Si c'est le cas, rejette cet échantillon. Sinon, attribue à l'échantillon la classe ayant la probabilité a posteriori maximale.

Pour score(self, X, y)

  • Prédit les classes pour X.
  • Identifie les observations qui n'ont pas été rejetés.
  • Compte combien d'entre eux ont été correctement classés et divise cette valeur par le nombre total d'observations non rejetés pour obtenir le score.
import numpy

class ClassifieurAvecRejet:
    def __init__(self, _lambda=1):
        """
        FR: Initialisez votre classifieur avec le paramètre lambda, le coût de rejet.
            Cette fonction ne doit pas être modifiée.

            Args:
                _lambda (float): Le coût de rejet.

        EN: Initialize your classifier with the parameter lambda, the cost of rejection.
            This function should not be modified.

            Args:
                _lambda (float): The cost of rejection.
        """
        self._lambda = _lambda

        # Ces attributs sont initialisés lors de l'appel à `fit`
        # These attributes are initialized when calling `fit`
        self.d = self.classes = self.probs_c = self.means = self.vars = None

    def fit(self, X, y):
        """
        FR: Entraînez votre modèle à partir des données fournies en argument.

            Args:
                X (numpy.ndarray): Une table de données de la forme (n, d), où n est le
                    nombre d'exemples et d le nombre de caractéristiques.
                y (numpy.ndarray): Une table d'étiquettes de la forme (n,), où n est le
                    nombre d'exemples.

        EN: Train the model from datasets given as arguments

            Args:
                X (numpy.ndarray): An array of data of shape (n, d), where n is the
                    number of examples and d the number of features.
                y (numpy.ndarray): An array of labels of shape (n,), where n is the the
                    number of examples.
        """
        # Les classes uniques présentes dans y et combien de fois chaque classe
        # apparaît.
        self.classes, counts = numpy.unique(y, return_counts=True)
        # Probabilités a priori des classes obtenues en divisant le nombre d'occurrences
        # de chaque classe par le nombre total d'observations.
        self.probs_c = counts / len(y)
        # Obtenir le nombre de caractéristiques (D)
        self.d = X.shape[1]

        num_classes = len(self.classes)
        self.means = numpy.zeros((num_classes, self.d))
        self.vars = numpy.zeros(num_classes)

        # Calculer les moyennes et variances pour chaque classe
        for i, c in enumerate(self.classes):
            # X_c représente le sous-ensemble des données d'entraînement appartenant à la classe i.
            X_c = X[y == c]
            self.means[i] = numpy.mean(X_c, axis=0)
            # Pour chaque caractéristique de notre jeu de données,
            # numpy.var(X_c, axis=0) calcule la variance de chaque caractéristique pour la classe i.
            # Étant donné que notre matrice de covariance est isotropique, toutes les variances le
            # long de la diagonale de la matrice de covariance sont égales. Ainsi, nous prenons la
            # moyenne des variances calculées pour chaque caractéristique pour obtenir une seule valeur
            # $\sigma_i^2$ qui sera utilisée pour toutes les caractéristiques.
            self.vars[i] = numpy.mean(numpy.var(X_c, axis=0))

    def predict_proba(self, X):
        """
        FR: Retournez la probabilité d'appartenance à chaque classe
            pour les données passées en argument.
            Cette fonction peut supposer que fit() a préalablement été appelé.
            Indice: calculez les termes de l'équation de Bayes séparément.

            Args:
                X (numpy.ndarray): Une table de données de la forme (n, d), où n est le
                    nombre d'exemples et d le nombre de caractéristiques.

            Return (numpy.ndarray): Un tableau de probabilités de la forme (n, c), où n
                est le nombre d'exemples et c le nombre de classes.

        EN: Return the class conditional probabilities to each class according
            to the data given as argument.
            This function can assume that fit() has been previously called.
            Tip: compute each component of Bayes equation separately.

            Args:
                X (numpy.ndarray): An array of data of shape (n, d), where n is the
                    number of examples and d the number of features.

            Return (numpy.ndarray): An array of probabilities of shape (n, c), where n
                is the number of examples and c the number of classes.
        """
        probs = numpy.zeros((X.shape[0], len(self.classes)))

        for i, c in enumerate(self.classes):
            mean = self.means[i]
            var = self.vars[i]

            # Le terme ci-dessous calcule la log-vraisemblance pour une distribution normale.
            # La vraisemblance est P(x|C), c'est-à-dire la probabilité d'observer les données x étant donné la classe C.
            # La log-vraisemblance est log P(x|C).
            log_likelihood = -0.5 * numpy.sum(((X - mean) ** 2) / var + numpy.log(var) + numpy.log(2 * numpy.pi), axis=1)

            # Nous utilisons des logarithmes pour éviter les problèmes de sous-débordement numérique.
            probs[:, i] = log_likelihood

        # Conversion des log-probabilités en probabilités réelles
        probs = numpy.exp(probs - probs.max(axis=1, keepdims=True))

        # Multiplier par la probabilité a priori. La probabilité a priori est la probabilité initiale
        # de chaque classe avant de voir les données.
        # Elle est généralement dérivée de la distribution des classes dans l'ensemble d'entraînement.
        probs *= self.probs_c

        # Diviser par l'évidence pour obtenir la probabilité a posteriori.
        # La probabilité a posteriori est la probabilité mise à jour de chaque classe après avoir vu les données.
        # En pratique, cela normalise également les probabilités pour qu'elles somment à 1 pour chaque échantillon.
        probs /= probs.sum(axis=1, keepdims=True)

        return probs

    def predict(self, X):
        """
        FR: Retournez les prédictions de classes pour les données
            passées en argument.
            Cette fonction peut supposer que fit() a préalablement été appelé.
            Indice: vous pouvez appeler predict_proba() pour éviter une
            redondance du code.

            Args:
                X (numpy.ndarray): Une table de données de la forme (n, d), où n est le
                    nombre d'exemples et d le nombre de caractéristiques.

            Return (numpy.ndarray): Un tableau de prédictions de la forme (n,), où n
                est le nombre d'exemples.

        EN: Return the class predictions for the data pass as arguments.
            This function can assume that fit() has been previously called.
            Tip: you can call predict_proba() to avoid repeating some code.

            Args:
                X (numpy.ndarray): An array of data of shape (n, d), where n is the
                    number of examples and d the number of features.

            Return (numpy.ndarray): An array of predictions of shape (n,), where n
                is the number of examples.
        """
        probs = self.predict_proba(X)
        max_probs = numpy.max(probs, axis=1)

        # Rejeter si la probabilité maximale est inférieure à 1 - lambda
        predictions = numpy.where(max_probs < (1 - self._lambda), -1, numpy.argmax(probs, axis=1))

        return predictions

    def score(self, X, y):
        """
        FR: Retournez le score de performance, tenant compte des données rejetées
            si lambda < 1.0, pour les données passées en argument.
            Cette fonction peut supposer que fit() a préalablement été exécuté.

            Args:
                X (numpy.ndarray): Une table de données de la forme (n, d), où n est le
                    nombre d'exemples et d le nombre de caractéristiques.
                y (numpy.ndarray): Une table d'étiquettes de la forme (n,), où n est le
                    nombre d'exemples.

            Return (float): Le score de performance.

        EN: Return the performance score, taking into account rejected samples
            if lambda < 1.0, for data given as argument.
            This function can assume that fit() has been previously called.

            Args:
                X (numpy.ndarray): An array of data of shape (n, d), where n is the
                    number of examples and d the number of features.
                y (numpy.ndarray): An array of labels of shape (n,), where n is the the
                    number of examples.

            Return (float): The performance score.
        """
        y_pred = self.predict(X)
        accepted_indices = y_pred != -1

        # Ne considérer que les observations qui n'ont pas été rejetés
        correct = numpy.sum(y[accepted_indices] == y_pred[accepted_indices])

        return correct / numpy.sum(accepted_indices)
