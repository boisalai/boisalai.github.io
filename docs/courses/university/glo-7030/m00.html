Module 0 : Apprentissage automatique, probabilités

Préalables​

Nous prenons pour acquis que vous avez une certaine connaissance de la plupart des concepts de base de l'apprentissage automatique, d'algèbre linéaire, et de probabilités. Pour ceux qui ont fait les préalables, vous devriez déjà avoir ces bases. Pour les autres, vous devrez déployer possiblement des efforts supplémentaires pour acquérir ces fondements, sur votre propre temps. Les concepts fondamentaux importants pour ce cours sont les suivants :

  • le produit scalaire et de matrice;
  • l'apprentissage supervisé, semi-supervisé et non-supervisé;
  • le sur-apprentissage (overfitting);
  • la classification vs la régression;
  • les classificateurs classiques comme k-NN, SVM, fonction logistique;
  • une idée approximative de qu'est-ce que la complexité (puissance) d'un classificateur;
  • ensembles d'entraînement, de validation, et de test;
  • la validation croisée;
  • les méthodes de réduction de la dimensionnalité (PCA);
  • les priors.

A symmetric matrix is a square matrix that is equal to its transpose: A=A⊤\boldsymbol{A} = \boldsymbol{A}^\topA=A⊤.

A Unit vector is a vector with unit norm: ∣∣x∣∣2=1|| \boldsymbol{x} ||_2 = 1∣∣x∣∣2​=1.

A vector is orthogonal to another vector if their dot product is zero: x⊤y=0\boldsymbol{x}^\top\boldsymbol{y} = 0x⊤y=0.

An orthogonal matrix is a square matrix whose rows are mutually orthonormal and whose columns are mutually orthonormal: A⊤A=AA⊤=I\boldsymbol{A}^\top\boldsymbol{A} = \boldsymbol{A}\boldsymbol{A}^\top = \boldsymbol{I}A⊤A=AA⊤=I. This implies that A−1=A⊤\boldsymbol{A}^{-1} = \boldsymbol{A}^\topA−1=A⊤.

An eigenvector of a square matrix A\boldsymbol{A}A is a non-zero vector that, when the matrix is multiplied by it, yields a constant multiple of itself, the multiplier being commonly denoted by λ\lambdaλ (the eigenvalue associated with the eigenvector): Av=λv\boldsymbol{A}\boldsymbol{v} = \lambda\boldsymbol{v}Av=λv. The scalar λ\lambdaλ is called the eigenvalue of A\boldsymbol{A}A corresponding to v\boldsymbol{v}v.

Two random variables xxx and yyy are independent if their probability distribution can be expressed as a product of two factors, one involving only xxx and the other involving only yyy: p(x,y)=p(x)p(y)p(x,y) = p(x)p(y)p(x,y)=p(x)p(y).

Two random variables xxx and yyy are conditionally independent given a random variable zzz if the conditional probability distribution over xxx and yyy factorizes in this way for every value of zzz: p(x,y∣z)=p(x∣z)p(y∣z)p(x,y|z) = p(x|z)p(y|z)p(x,y∣z)=p(x∣z)p(y∣z).

The kernel trick is a method of using a linear classifier to solve a non-linear problem. It transforms the linearly inseparable data into a higher dimension so that it becomes linearly separable.

The kernel is a function that takes two vectors as input and outputs the dot product of the vectors.
