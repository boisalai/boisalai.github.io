GIF-7005 Introduction à l'apprentissage automatique

Informations générales​

  • Enseignant : Christian Gagné.
  • Cours comodal, du 6 sept. 2023 au 13 déc. 2023.
  • Fiche.
  • 🎥 Youtube playlists.
  • Portail.
  • Site PAX (notebook et forum).

Ce cours porte sur les méthodes permettant l'inférence à partir d'observations de modèles de classement, de régression, d'analyse de données ou de prise de décision. Ces méthodes se caractérisent par une phase d'entraînement à partir de données ou d'expériences, afin d'effectuer des tâches qui seraient difficiles ou impossibles à faire par des moyens algorithmiques plus classiques. Le cours aborde différentes approches actives en apprentissage et cherche à expliquer leurs mécanismes de base. Une perspective applicative de ces différentes techniques est également présentée, avec un accent particulier sur l'utilisation d'outils logiciels modernes.

Matériel didactique :

  • 📕 Ethem Alpaydin, Introduction to Machine Learning (4e édition), MIT Press (2020).
  • 📙 Bishop, Christopher M, Pattern Recognition and Machine Learning, Springer (New York, 2006) (disponible sur ce site).
  • 📗 Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani, Jonathan Taylor, An Introduction to Statistical Learning with Applications in Python (2e édition), Springer (Cham, 2023) (disponible sur ce site).
  • Ian Goodfellow, Yoshua Bengio et Aaron Courville, Deep Learning, MIT Press (Boston, 2016) (disponible sur ce site).
  • Kevin P. Murphy, Probabilistic Machine Learning : An Introduction Kevin P. Murphy, The MIT Press (Cambridge, Mass., 2022) (disponible sur ce site).
  • Andriy Burkov, The Hundred-Page Machine Learning Book, 2019 (disponible sur ce site).
  • Aurélien Géron, Hands-on machine learning with Scikit-Learn, Keras, and TensorFlow : concepts, tools, and techniques to build intelligent systems (2e édition), 2019, Version électronique.
  • Richard O. Duda, David G. Stork and Peter E. Hart, Pattern classification and scene analysis. (2e édition), Wiley (New York, 2000).

convention

Les traductions accuracy = exactitude et precision = précision sont les termes à utiliser. Mais l'instructeur dit ceci : Oui, ce sont les bons termes, mais étant donné la confusion possible, j'essaie souvent d'éviter d'utiliser accuracy, je parle plutôt de classification rate = taux de classement pour décrire le même concept.

Bonne ressource pour les traductions français/anglais des termes dans le domaine.

Par ailleurs, il faut remplacer le chemin /pax/shared/GIF-4101-7005/ par https://pax.ulaval.ca/static/GIF-4101-7005/fichiers/ pour en faire un URL. Par exemple, si vous voulez télécharger le fichier /pax/shared/GIF-4101-7005/d2q2_datasets/public1/features.npy, l'URL correspondant sera https://pax.ulaval.ca/static/GIF-4101-7005/fichiers/d2q2_datasets/public1/features.npy.
