Travail pratique 3

  • L'énoncé du dernier travail est disponible.
    • Thème : Analyse d'incidents en utilisant un modèle de langue (LLM) ou un modèle encodeur-décodeur.
  • Date limite pour ceux qui veulent proposer un sujet libre: le 1er décembre.
    • Déposez votre proposition dans la boîte de remise de votre équipe (en format PDF).
    • Je pourrai vous confirmer suite au dépôt de votre proposition.
  • Date de remise:
    • Idéalement avant le vendredi 16 décembre.
    • Obligatoirement avant la fin de session d'automne. Me consulter au besoin.

Objectif​

  • Utiliser des modèles transformers préentraînés (donc, plus de un modèle)
  • Obtenir des informations sur des indidents à partir de descriptions textuelles
  • Comprendre comment obtenir les informations en utilisant un nombre limité d'exemples
  • Évaluer la performance des modèles
  • Enrichir le jeu de données en annotant 10 nouveaux exemples
